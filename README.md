Nome do sistema

Vigilante

Objetivo

O sistema Vigilante foi desenvolvido para a Mostra STEAM com o objetivo de ajudar na conscientização sobre segurança digital e proteção de dados.

O projeto permite cadastrar informações, visualizar registros, atualizar dados e excluir informações através da integração entre Front-End, Back-End e banco de dados.

Tecnologias utilizadas
HTML

CSS

JavaScript

Node.js

Express

MySQL

Fetch API

GitHub

Como executar

-baixar o projeto pelo Git Hub

- Abrir a pasta no vs code

- E por o codigo para rodar com o Node no Terminal
